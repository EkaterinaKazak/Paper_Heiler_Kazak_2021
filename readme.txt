Readme file for the paper

Valid inference for treatment effect parameters under irregular identification and many extreme propensity scores

Heiler and Kazak, 2021

https://doi.org/10.1016/j.jeconom.2020.03.025

The following folders contain replication Files for Sections 5 and 6.

ReplicationFiles_Section5 contains the simulation code (MATLAB) for IPW and DR in Section 5. The file 'an_ipw_example.m' and 'a_dr_example.m' can be executed for replication of the results in tables 5.1 - 5.3. 

ReplicationFiles_Section6 contains the R code for the application in Section 6. 

Explicit comments for execution can be found within the files. For further questions please contact pheiler@econ.au.dk or ekaterina.kazak@manchester.ac.uk
